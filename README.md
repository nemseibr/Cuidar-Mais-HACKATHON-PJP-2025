## 🎯 Sobre o Projeto

O **Cuidar Mais** é uma aplicação web dedicada a proporcionar acolhimento e cuidado de qualidade. 
A plataforma integra diversos recursos para apoiar usuários em suas necessidades de saúde e bem-estar, incluindo:

- 📚 **Orientações Práticas**: Informações claras sobre cuidados e rotinas diárias
- 💬 **Canal de Comunicação**: Comunicação acolhedora e empática
- ⚖️ **Suporte Jurídico**: Documentos e orientações legais simplificadas
- 👥 **Gestão de Equipe**: Organização da equipe de cuidadores
- 📋 **Histórico de Pacientes**: Rastreamento de informações do paciente
- 📞 **Teleatendimento**: Atendimento remoto e acessível
- 🏥 **Acolhimento**: Interface dedicada ao acolhimento inicial
- ⚙️ **Configurações**: Personalização da plataforma

🛠️ Tecnologias Utilizadas

🌐 Frontend (o que o usuário vê)
Utilizamos o Next.js junto com React para construir a interface da aplicação
O projeto foi desenvolvido com TypeScript, que ajuda a evitar erros no código
Para o visual, usamos Tailwind CSS, que facilita a criação de layouts modernos e responsivos

🎨 Interface e Componentes
Utilizamos o Radix UI para criar componentes acessíveis e bem estruturados
Os ícones vêm do Lucide React
As notificações do sistema são feitas com Sonner

📝 Formulários
Usamos React Hook Form para lidar com formulários de forma simples e eficiente
A validação dos dados é feita com Zod, garantindo que tudo seja preenchido corretamente

⚙️ Funcionalidades Extras
date-fns para trabalhar com datas
Recharts para exibir gráficos
Componentes como carrossel, painéis ajustáveis e suporte a tema claro/escuro também foram implementados

🚀 Build e Preparação do Projeto
Utilizamos ferramentas como PostCSS e Autoprefixer para garantir compatibilidade entre navegadores
